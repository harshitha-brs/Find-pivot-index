Find Pivot Index



The pivot index is the index where the sum of all the numbers strictly to the

left of the index is equal to the sum of all the numbers strictly to the index's right.

If the index is on the left edge of the array, then the left

sum is 0 because there are no elements to the left.

Input: nums = [1,7,3,6,5,6] Output: 3

Input: nums = [1,2,3] Output: -1

Input: nums = [2,1,-1] Output: 0
